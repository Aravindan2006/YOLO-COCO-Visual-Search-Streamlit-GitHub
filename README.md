# YOLO-COCO-Visual-Search-Streamlit-GitHub


### A Streamlit-based visual search application using YOLO and the COCO dataset for real-time object detection and image similarity search.

## Features
```
YOLO object detection
COCO dataset support
Image upload and search
Streamlit web interface
Fast visual search results
```
## Technologies
```
Python
YOLO
OpenCV
Streamlit
NumPy
```
## Run the Project
```
pip install -r requirements.txt
streamlit run app.py
```
